Глобальный доступ

Простая настройка за 5 минут

Следуйте этим шагам и начните управлять музыкой

1

Запустите Яндекс.Музыку с параметрами

Откройте Яндекс.Музыку с дополнительными параметрами для удаленного доступа

--remote-debugging-port=9222 --remote-debugging-address=0.0.0.0
2

Скачайте и запустите сервер

Скачайте архив с сервером и запустите его одним кликом

# Windows start.bat # Linux/Mac ./start.sh
3

Откройте интерфейс управления

Перейдите по ссылке в браузере или отсканируйте QR код с телефона

http://localhost:3002
4

Настройте глобальный доступ (опционально)

Для доступа из интернета настройте проброс портов на роутере

Порт 3002 (TCP) → ваш компьютер Порт 3003 (TCP) → ваш компьютер
